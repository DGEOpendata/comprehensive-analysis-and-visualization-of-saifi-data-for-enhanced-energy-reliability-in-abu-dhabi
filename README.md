# Comprehensive Analysis and Visualization of SAIFI Data

This project provides a Python-based solution for analyzing and visualizing the 'Annual SAIFI (System Average Interruption Frequency Index) Data (2013-2025)' dataset. The goal is to help energy professionals, researchers, and policymakers gain actionable insights from the data and improve power reliability in Abu Dhabi.

## Features

1. **Data Visualization:** Visualize year-over-year changes in SAIFI values.
2. **Trend Analysis:** Understand historical trends in power interruptions.
3. **Predictive Analytics:** Forecast SAIFI values for future years using linear regression.
4. **Custom Reporting:** Generate detailed reports tailored to various stakeholders.

## Prerequisites

- Python 3.8 or higher
- Required libraries: `pandas`, `matplotlib`, `sklearn`, `numpy`

## Getting Started

1. Clone the repository:
   bash
   git clone https://github.com/your-repository/saifi-analysis.git
   cd saifi-analysis
   

2. Install dependencies:
   bash
   pip install -r requirements.txt
   

3. Download the dataset:
   - Save the `SAIFI.csv` file in the root directory of the project.

4. Run the analysis script:
   bash
   python analyze_saifi.py
   

5. View the generated visualizations and predictive analysis results.

## Expected Output

- Line chart of historical SAIFI values (2013-2025).
- Line chart of historical and predicted SAIFI values (2013-2030).
- Insights into year-over-year trends in power reliability.
